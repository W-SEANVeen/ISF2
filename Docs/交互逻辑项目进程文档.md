# 项目进程文档 — 交互逻辑部分

> 基于 `Assets/Script/` 目录下全部脚本的反向整理，描述各系统间的交互关系和数据流向。

---

## 一、整体架构概览

整个交互逻辑围绕一场**VR攻城战**展开，玩家扮演守城统帅，敌军按波次进攻。系统分为三大模块：

```
┌─────────────────────────────────────────────────────┐
│                   BattleDirector                     │  ← 总导演：控制战斗节奏与波次
├─────────────────────────────────────────────────────┤
│   ┌──────────┐  ┌───────────┐  ┌─────────────────┐ │
│   │ SiegeMgr │  │  Player   │  │   ArrowRain      │ │  ← 平行系统
│   │ (推进)   │  │  (玩家)   │  │   Volley (箭雨)  │ │
│   └────┬─────┘  └─────┬─────┘  └────────┬────────┘ │
│        │               │                  │          │
│   ┌────▼───────────────▼──────────────────▼────────┐ │
│   │              BattleDirector                     │ │
│   │  管理波次生成 → 监控梯子 → 释放精英死士         │ │
│   └────────────────────┬───────────────────────────┘ │
│                        │                             │
│   ┌────────────────────▼───────────────────────────┐ │
│   │          SquadCommander + ArmyMover             │ │
│   │  生成小兵方阵 → 维护阵型 → 抵达城墙 → 架梯子   │ │
│   └────────────────────┬───────────────────────────┘ │
│                        │                             │
│   ┌────────────────────▼───────────────────────────┐ │
│   │         SiegeLadder → EnemyAssault              │ │
│   │    梯子竖起 → 通知导演 → 精英死士攀爬 → 追杀   │ │
│   └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## 二、核心交互流程

### 2.1 战斗启动链路

```
Game Start
    │
    ▼
BattleDirector.Awake()          → 注册单例 Instance
    │
    ▼
BattleDirector.Start()
    │  Invoke("StartBattle", 3f)  ← 延迟 3 秒开战
    ▼
BattleDirector.StartBattle()
    │  isBattleActive = true
    ├─ StartCoroutine(ArmyChargeRoutine())    → 波次生成循环
    └─ StartCoroutine(ArrowRainRoutine())     → 箭雨定时器
```

**关键数据**：
- `currentWave`：当前波次编号（从 1 开始）
- `currentActiveEnemies`：场上存活敌人数
- `maxEnemiesOnField`：场上最大兵力上限（默认 30）

### 2.2 波次生成 → 小兵行军 → 架梯链路

```
ArmyChargeRoutine (循环)
    │  if (currentActiveEnemies >= maxEnemiesOnField) → 等待
    │  否则每 waveInterval(25s) 生成一波
    ▼
Instantiate(prefab, spawnPoint)
    │
    ├─ SquadCommander.ReceiveOrders(targetWall)
    │      agent.SetDestination(targetWall.position)
    │
    ├─ siegeManager.globalDamping 影响 Agent.speed  ← 核心压制交互
    │
    └─ attachedLadder → 标记所属梯子
```

#### 阵型维护 (`SquadCommander.MaintainFormation()`)

```
Update 每帧
    │
    ├─ agent.speed = baseSpeed * siegeManager.globalDamping
    │                                  ↑
    │                             受玩家压制影响
    │
    └─ MaintainFormation()
           │
           ├─ 计算"绝对基准轴" = 指向城墙的方向（不受转弯影响）
           │
           ├─ 为每个 squadMember 记录 localOffsets（相对队长位置）
           │
           ├─ 目标位置 = 队长位置 + 基准旋转 × 初始偏移
           │
           ├─ 防抖机制：仅当目的地偏离 >1m 时才重新 SetDestination
           │
           └─ 追赶机制：掉队 >3m 时加速到 personalSpeed
```

#### 抵达 → 架梯 → 工具人撤离

```
agent.remainingDistance <= stoppingDistance
    │
    ├─ hasArrived = true           ← 防重复触发锁
    ├─ agent.enabled = false       ← 关掉 NavMesh 节省性能
    ├─ transform.position = 城墙坐标（吸附对齐）
    │
    ├─ attachedLadder.StartErecting()
    │      └─ SiegeLadder.Update()
    │            └─ 每帧 Lerp 旋转 → 达到 targetAngle(75°) 时完成
    │                  └─ LadderReady()
    │                        └─ BattleDirector.OnLadderErected(this)
    │
    └─ portersGroup.StartRetreat()
           └─ PorterRetreat
                 ├─ SetParent(null)  ← 脱离父子层级
                 ├─ 工具人随机朝向 120°~240° 逃跑
                 └─ Destroy(gameObject, 3s)  ← 性能回收
```

### 2.3 精英死士释放链路（第二阶段核心交互）

```
BattleDirector.OnLadderErected(SiegeLadder reporter)
    │
    ├─ 检查 hasDeployedElite          → true 则拦截（防重复释放）
    ├─ 检查 theOneAndOnlyElite        → null 则报错
    ├─ 检查 routeLadders 数组有效性   → 索引越界则报错
    ├─ 检查 routeLadders[designatedEliteRouteIndex] → 空则报错
    │
    └─ reporter == expectedLadder?
           │
           ├─ ✅ 匹配成功 → theOneAndOnlyElite.StartAssaultWithTarget(anchor)
           │                   └─ EnemyAssault 状态机启动
           │
           └─ ❌ 匹配失败 → 不释放，等待下一把梯子
```

### 2.4 精英死士状态机

```
EnemyAssault 状态流转：

    Idle
     │  StartAssaultWithTarget()
     ▼
    RunningToLadder
     │  agent.SetDestination(bottomAnchor.position)
     │  OnTriggerEnter(BottomTrigger)
     ▼
    Climbing
     │  agent.enabled = false, rb.isKinematic = true
     │  transform.Translate(Vector3.up * climbSpeed)
     │  OnTriggerEnter(TopTrigger)
     ▼
    ClimbingEnd
     │  anim.SetTrigger("ClimbEnd")
     │  WaitForSeconds(climbEndTime=1.5s)
     │  transform.position = topAnchor.position
     │  agent.enabled = true
     ▼
    ChasingPlayer
     │  agent.SetDestination(playerTransform.position)
     │  if 距离 ≤ attackDistance(1.5m) → StartAttack()
     ▼
    Attacking
     │  agent.isStopped = true
     │  anim.SetTrigger("Attack")
     │  WaitForSeconds(3s) → 恢复 ChasingPlayer
     │
     │  动画事件 → EnableWeaponTrail() / DisableWeaponTrail()
     ▼
    ChasingPlayer (循环)
```

**武器拖尾协作**：
```
EnemyAssault                    WeaponTrailController
     │                                  │
     ├─ EnableWeaponTrail()  ──────────► myTrail.Clear() + emitting = true
     ├─ DisableWeaponTrail() ──────────► emitting = false
     └─ ResetWeaponTrail()   ──────────► Clear() + emitting = false
```

### 2.5 箭雨系统

```
ArrowRainRoutine (BattleDirector 协程)
    │  首次延迟 firstArrowDelay(5s)
    │  之后每 arrowRainInterval(15s) ± 3s 随机间隔
    ▼
ArrowRainVolley.FireArrowRain()
    │
    └─ SpawnAndLaunchArrows()
           │
           ├─ 从 sourceFiringArea (BoxCollider) 随机起点
           │
           ├─ 射线检测：从 skyCloudBox 随机点向下发射 → 命中 wallLayer 得到目标点
           │
           ├─ 计算抛物线速度 (CalculateLaunchVelocity)
           │     └─ 已知起点、终点、飞行时间 → 反推初速度
           │
           └─ 对象池管理 (Queue<GameObject>)
                 ├─ Awake() 预生成 poolSize(250) 支箭
                 ├─ GetArrowFromPool() → 取出/扩容
                 └─ ReturnArrowToPool() → 回收

箭矢 (ArrowFlight)：
    OnEnable → Clear Trail, 重置物理状态, Invoke Recycle(4~7s)
    Update   → 根据 velocity 旋转箭身
    OnCollisionEnter → isKinematic = true (插在墙上)
    OnDisable → CancelInvoke (防报错)
```

### 2.6 玩家系统

#### 中箭 → 僵直 → 压制反馈

```
PlayerHealth.OnTriggerEnter(Collider other)
    │  other.CompareTag("EnemyArrow") && !isStunned
    ├─ TakeDamage(10)
    │     └─ currentHP -= 10; <=0 → 城破（TODO）
    │
    ├─ Destroy(arrow)              ← 销毁箭矢
    │
    └─ StartCoroutine(StunRoutine())
           │
           ├─ isStunned = true
           ├─ locomotionSystem.enabled = false     ← 禁摇杆移动
           ├─ leftHand.enableInputActions = false  ← 禁左手按键（开火）
           ├─ rightHand.enableInputActions = false ← 禁右手按键
           │    ★ 双手追踪仍然生效，仅禁用按钮
           │
           ├─ WaitForSeconds(1s)
           │
           ├─ locomotionSystem.enabled = true      ← 恢复
           ├─ leftHand.enableInputActions = true
           ├─ rightHand.enableInputActions = true
           └─ isStunned = false
```

#### 大黄弩射击

```
CrossbowShoot.Fire()
    │  (VR 手柄按键触发)
    ├─ Instantiate(arrowPrefab, firePoint)
    ├─ Rigidbody.AddForce(firePoint.forward * shootForce(2000))
    └─ Destroy(arrow, 5s)
```

**核心交互**: 玩家的弩箭命中敌人后 → `EnemyHealth.Die()` → `SiegeManager.TriggerSuppressingFire()` → `globalDamping = 0.1` → 全军行军速度骤降。

### 2.7 火把/燃烧系统

```
TorchFireController (火把)
    │  Start 中通过代码配置 ParticleSystem
    │  提供 Ignite() / Extinguish() 接口
    │
    对手持 Igniter 标签的物体触发：
    RickIgniteController.OnTriggerEnter(Collider other)
        │  other.CompareTag("Igniter") && !isOnFire
        └─ IgniteRoutine(contactPoint)
               │
               ├─ Phase 1: Instantiate(startFirePrefab) → 小火苗
               │
               ├─ WaitForSeconds(smokeDelay=1.2s)
               │
               ├─ Phase 2: Instantiate(mainSmokePrefab) → 大浓烟
               │
               ├─ Phase 3: ps.Stop(StopEmitting) → 小火消散
               │
               └─ Destroy(currentStartFire, 2s) → 性能回收
```

## 三、跨系统数据流汇总

### 3.1 全局阻尼压制流 (SiegeManager)

```
玩家击杀敌人
    │
    ▼
EnemyHealth.Die()
    │
    ├─ siegeManager.TriggerSuppressingFire()
    │      └─ globalDamping = 0.1    ← 瞬间压制
    │
    └─ (2s 后) OnDestroy()
           └─ BattleDirector.currentActiveEnemies--
                 └─ 腾出兵力名额，下一波可继续生成

SiegeManager.Update()
    ├─ siegeProgress += baseAdvanceSpeed * globalDamping * Time.deltaTime
    └─ globalDamping 缓慢恢复至 1.0

被影响方：
    SquadCommander:
        agent.speed = baseSpeed * siegeManager.globalDamping
    ├─ 直接影响队长移动
    └─ 队长挪不动 → 小兵阵型卡在原地
```

### 3.2 兵力计数流

```
EnemyHealth.Start()
    └─ BattleDirector.Instance.currentActiveEnemies++

EnemyHealth.OnDestroy()
    └─ BattleDirector.Instance.currentActiveEnemies--

BattleDirector.ArmyChargeRoutine()
    └─ currentActiveEnemies >= maxEnemiesOnField → 暂停出兵
```

### 3.3 梯子-死士释放流

```
BattleDirector 数组:
    routeLadders[i] = SquadCommander.attachedLadder ← 第 i 路梯子

SiegeLadder.LadderReady()
    └─ BattleDirector.OnLadderErected(this)
           └─ routeLadders[designatedEliteRouteIndex] == this?
                  → 匹配成功 → EnemyAssault.StartAssaultWithTarget()

EnemyAssault 攀爬全程:
    StartAssaultWithTarget(anchor) → RunningToLadder → Climbing → ClimbingEnd → ChasingPlayer
```
